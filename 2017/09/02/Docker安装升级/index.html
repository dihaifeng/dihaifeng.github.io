Dockerе®үиЈ…дёҺеҚҮзә§

еҸ‘иЎЁдәҺ 2017-09-02

Centos 7 е®үиЈ…Docker-CE(зӨҫеҢәзүҲ)

зі»з»ҹиҰҒжұӮ

иҰҒе®үиЈ…Docker CEпјҢйңҖиҰҒ64дҪҚзүҲжң¬зҡ„CentOS7зі»з»ҹгҖӮйңҖиҰҒеҗҜз”Ёcentos-extrasеә“пјҢй»ҳи®Өжғ…еҶөе–ңзҲұжӯӨеӯҳеӮЁеә“дҪҝеҗҜз”Ёзҡ„пјҢеҰӮжһңе·ІзҰҒз”ЁеҲҷйңҖиҰҒйҮҚж–°еҗҜз”Ё

еҚёиҪҪж—§зүҲжң¬

иҫғиҖҒзүҲжң¬зҡ„Dockerиў«з§°дёәdockerжҲ–docker-engineгҖӮеҰӮжһңиҝҷдәӣе·Із»Ҹе®үиЈ…пјҢиҜ·еҚёиҪҪе®ғд»¬д»ҘеҸҠе…іиҒ”зҡ„дҫқиө–е…ізі»гҖӮ

е®үиЈ…Docker CE

е®үиЈ…Docker CEжңүеҮ з§ҚдёҚеҗҢзҡ„ж–№ејҸпјҡ

  • д»ҺDockerзҡ„еӯҳеӮЁеә“дёӯе®үиЈ…пјҢж–№дҫҝе®үиЈ…е’ҢеҚҮзә§д»»еҠЎгҖӮе®ҳзҪ‘жҺЁиҚҗ
  • дёӢиҪҪRPMиҪҜд»¶е№¶жүӢеҠЁе®үиЈ…пјҢжүӢеҠЁе®ҢжҲҗз®ЎзҗҶеҚҮзә§гҖӮиҝҷз§Қж–№жі•з”ЁдәҺжІЎжңүи®ҝй—®дә’иҒ”зҪ‘зҡ„зҺҜеўғдёӢдҪҝз”Ё
  • еңЁжөӢиҜ•е’ҢејҖеҸ‘зҺҜеўғдёӯпјҢеҸҜд»ҘйҖүжӢ©е®ҳзҪ‘жҸҗдҫӣзҡ„иҮӘеҠЁеҢ–е®үиЈ…и„ҡжң¬
дҪҝз”ЁеӯҳеӮЁеә“е®үиЈ…

еңЁж–°дё»жңәдёҠйҰ–ж¬Ўе®үиЈ…Docker CEд№ӢеүҚпјҢйңҖиҰҒи®ҫзҪ®DockerеӯҳеӮЁеә“гҖӮд№ӢеҗҺпјҢе°ұеҸҜд»Ҙд»ҺеӯҳеӮЁеә“е®үиЈ…е’Ңжӣҙж–°DockerгҖӮ

  • е®үиЈ…жүҖйңҖиҪҜд»¶еҢ…yum-utilsжҸҗдҫӣдәҶyum-config-managerеҠҹиғҪпјҢdevice-mapper-persistent-dataе’Ңlvm2 жҸҗдҫӣдәҶdevicemapperеӯҳеӮЁй©ұеҠЁзЁӢеәҸ
  • дҪҝз”Ёд»ҘдёӢе‘Ҫд»Өи®ҫзҪ®зЁіе®ҡзүҲжң¬еә“
  • еҸҜйҖүпјҡеҗҜз”Ёedgeе’Ңtestеә“пјҢиҝҷдәӣеӯҳеӮЁеә“еҢ…еҗ«еңЁdocker.repoж–Үд»¶дёӯпјҢдҪҶй»ҳи®Өжғ…еҶөдёӢжҳҜзҰҒз”Ёзҡ„пјҢеҸҜд»Ҙе°Ҷе®ғд»¬дёҺзЁіе®ҡзүҲжң¬еә“дёҖиө·еҗҜз”Ё
  • е®үиЈ…Docker CE
д»ҺеҢ…дёӯе®үиЈ…

еҰӮжһңдёҚиғҪдҪҝз”ЁDockerзҡ„еӯҳеӮЁеә“е®үиЈ…DockerпјҢеҸҜд»ҘдёӢиҪҪ.rpmзүҲжң¬зҡ„ж–Үд»¶е№¶жүӢеҠЁе®үиЈ…гҖӮжҜҸж¬ЎиҰҒеҚҮзә§Dockerж—¶пјҢйғҪйңҖиҰҒдёӢиҪҪдёҖдёӘж–°ж–Үд»¶гҖӮ

  • иҪ¬еҲ° https://download.docker.com/linux/centos/7/x86_64/stable/Packages/ е№¶дёӢиҪҪ.rpmиҰҒе®үиЈ…зҡ„DockerзүҲжң¬зҡ„ж–Үд»¶гҖӮ
  • е®үиЈ…Docker CE пјҢе°ҶдёӢйқўзҡ„и·Ҝеҫ„жӣҙж”№дёәе·Із»ҸдёӢиҪҪDockerзЁӢеәҸеҢ…зҡ„и·Ҝеҫ„пјҢе®үиЈ…е№¶еҗҜеҠЁ
  • еҚҮзә§Docker CE
    иҰҒеҚҮзә§Docker CEпјҢйңҖиҰҒдёӢиҪҪиҫғж–°зҡ„иҪҜд»¶еҢ…йҘјйҮҚеӨҚе®үиЈ…пјҢдҪҝз”Ёyum -y upgradeжӣҙж–°е®үиЈ…
еҲ©з”Ёи„ҡжң¬иҝӣиЎҢе®үиЈ…

DockerеңЁget.docker.com е’Ңtest.docker.comдёҠжҸҗдҫӣдәҶиҮӘеҠЁеҢ–е®үиЈ…и„ҡжң¬пјҢз”ЁдәҺе°ҶDocker CEзҡ„зЁіе®ҡе’ҢжөӢиҜ•зүҲжң¬еҝ«йҖҹйғЁзҪІпјҢйқһдәӨдә’ејҸе®үиЈ…еҲ°ејҖеҸ‘зҺҜеўғдёӯгҖӮи„ҡжң¬зҡ„жәҗд»Јз ҒдҪҚдәҺ docker-installеӯҳеӮЁеә“дёӯгҖӮ дёҚе»әи®®еңЁз”ҹдә§зҺҜеўғдёӯдҪҝз”Ёиҝҷдәӣи„ҡжң¬ пјҢдҪҝз”Ёиҝҷдәӣи„ҡжң¬е…·жңүдёҖе®ҡжҪңеңЁзҡ„йЈҺйҷ©пјҡ

  • и„ҡжң¬йңҖиҰҒrootжҲ–sudoжқғйҷҗжүҚиғҪиҝҗиЎҢгҖӮеӣ жӯӨпјҢеңЁиҝҗиЎҢи„ҡжң¬д№ӢеүҚпјҢеә”д»”з»ҶжЈҖжҹҘе’Ңе®Ўж ёи„ҡжң¬гҖӮ
  • и„ҡжң¬е°қиҜ•жЈҖжөӢжӮЁзҡ„LinuxеҸ‘иЎҢзүҲжң¬е’ҢзүҲжң¬пјҢе№¶дёәжӮЁй…ҚзҪ®жӮЁзҡ„иҪҜд»¶еҢ…з®ЎзҗҶзі»з»ҹгҖӮжӯӨеӨ–пјҢи„ҡжң¬дёҚе…Ғи®ёжӮЁиҮӘе®ҡд№үд»»дҪ•е®үиЈ…еҸӮж•°гҖӮиҝҷеҸҜиғҪеҜјиҮҙдёҚеҸ—ж”ҜжҢҒзҡ„й…ҚзҪ®пјҢж— и®әжҳҜд»ҺDockerзҡ„и§’еәҰиҝҳжҳҜд»ҺжӮЁиҮӘе·ұзҡ„з»„з»Үзҡ„жҢҮеҜјж–№й’Ҳе’Ңж ҮеҮҶгҖӮ
  • и„ҡжң¬е®үиЈ…еҢ…з®ЎзҗҶеҷЁзҡ„жүҖжңүдҫқиө–е…ізі»е’Ңе»әи®®пјҢиҖҢдёҚиҰҒжұӮзЎ®и®ӨгҖӮиҝҷеҸҜиғҪдјҡе®үиЈ…еӨ§йҮҸзҡ„иҪҜд»¶еҢ…пјҢе…·дҪ“еҸ–еҶідәҺжӮЁзҡ„дё»жңәзҡ„еҪ“еүҚй…ҚзҪ®гҖӮ

  • еҰӮжһңDockerе·Із»ҸдҪҝз”Ёе…¶д»–жңәеҲ¶е®үиЈ…еңЁдё»жңәдёҠпјҢеҲҷдёҚиҰҒдҪҝз”ЁдҫҝеҲ©и„ҡжң¬гҖӮ
